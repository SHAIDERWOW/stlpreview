# 🧊 Visualizador 3D v3.0

Um visualizador 3D prático, rápido e interativo que roda diretamente no seu navegador. Ideal para analisar modelos tridimensionais e peças para impressão 3D sem a necessidade de baixar ou instalar nenhum software.

🌍 **Acesse o site:** [Insira o link do seu site aqui, ex: https://seu-usuario.github.io/stlpreview/]

## ✨ Funcionalidades Principais

- **Múltiplos Formatos Suportados:** Compatível com arquivos `.STL`, `.3MF`, `.OBJ`, `.glTF` e `.glb`.
- **Drag & Drop (Arraste e Solte):** Arraste o arquivo do seu computador e solte diretamente na página para visualizar instantaneamente.
- **Estimador de Tempo de Impressão:** Simule o tempo gasto na impressora 3D configurando os parâmetros de:
  - Resolução (Altura da camada)
  - Preenchimento (Infill %)
  - Velocidade (mm/s)
- **Ferramenta de Medição:** Clique em dois pontos do modelo para descobrir a distância exata entre eles.
- **Modos de Exibição:** Alterne facilmente a visualização entre modelo **Sólido**, malha **Wireframe** ou visão em **Raio-X**.
- **Capturas de Tela (Screenshot):** Exporte a visualização atual em Full HD com opções de fundo transparente, branco ou preto.
- **Personalização de Interface:** Altere a cor do modelo renderizado em tempo real e escolha entre o **Modo Claro** e **Modo Escuro** (sua preferência é salva automaticamente).

## 🖱️ Como Utilizar

1. Abra o link do visualizador no seu navegador.
2. Escolha um modelo da lista suspensa ou **arraste um arquivo do seu computador** para o centro da tela.
3. **Navegação com o Mouse:**
   - **Botão Esquerdo:** Clique e segure para rotacionar o modelo.
   - **Botão Direito:** Clique e segure para mover (Pan) a câmera.
   - **Scroll (Roda do Mouse):** Aumente ou diminua o Zoom.
4. Utilize o painel de configurações na tela para acessar os controles de visualização e ferramentas.

## ⌨️ Atalhos de Teclado

Ganhe tempo utilizando os atalhos rápidos do visualizador:

- <kbd>R</kbd> : Resetar a câmera e centralizar o modelo
- <kbd>G</kbd> : Mostrar / Ocultar a grade de referência (Grid)
- <kbd>S</kbd> : Abrir o menu de Captura de Tela (Screenshot)
